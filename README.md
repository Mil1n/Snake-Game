# 🐍 Neon Snake — Modern Arcade Edition

Стильная и динамичная версия классической Snake Game: неоновые цвета, glassmorphism-интерфейс, плавный геймплей и автосохранение рекорда.

## ✨ Особенности

- 🎮 Современный визуальный стиль: neon + grid + glass UI.
- ⚡ Ускорение игры по мере роста счёта.
- 🧠 Сохранение рекорда в `localStorage`.
- ⏸️ Пауза/возобновление по кнопке и клавише `P`.
- ⌨️ Управление через `WASD` и стрелки.
- 📱 Адаптивный интерфейс (удобно на разных экранах).

## 🚀 Быстрый старт

Открой `index.html` в браузере.

Или локально через простой HTTP-сервер:

```bash
python3 -m http.server 8080
```

После этого перейди на: `http://localhost:8080`

## 🕹️ Управление

- `Space` — начать новую игру
- `P` — пауза / продолжить
- `WASD` или `↑ ↓ ← →` — движение

## 📁 Структура проекта

- `index.html` — разметка игры и HUD
- `styles.css` — оформление интерфейса и неон-стили
- `script.js` — игровая логика
